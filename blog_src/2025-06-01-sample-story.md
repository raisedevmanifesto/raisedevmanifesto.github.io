# Welcome to the RAISE Blog

This is a sample blog post written in markdown and stored in the `blog_src/` directory.

The RAISE Manifesto website now supports a blog workflow using markdown sources and AI-powered publishing. This post demonstrates how authors can draft stories, document insights, and share updates—all within a simple, reviewable, and auditable source format.

## Why Markdown?

- Easy to write and edit
- Supports review and version control
- Enables future automation (AI, formatting, summaries)

## Next Steps

Once this entry is ready for publication, an AI process will convert it into a styled HTML page for the `/blog/` section of the site. Drafts can remain here until they're ready to be shared with the world.
